Flask-Cloud-NDB
================

Flask-Cloud-NDB is an extension for `Flask`_ that adds support for
`Google Cloud NDB`_ to your application.


Installing
----------

Install and update using `pip`_:

.. code-block:: text

  $ pip install -U Flask-Cloud-NDB
